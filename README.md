# CRUD Cliente - Vue + Node

## Sistema desenvolvido em:
* Angular Vue
* Nodejs 
* Express 
* MongoDB


## Back-End
### Comandos
```shell
# Instalar dependencias
npm install

# Iniciar sistema
nodemon serve

```


## Front-End

```shell
# Instalar dependencias 
npm install

#Inicar ng server
npm run serve
```

### Criação de banco

* Criar uma conta no https://cloud.mongodb.com/
* Criar arquivo .env e inserir o username e senha: 

```shell
USER_DB=username
PASSWORD_DB=senha
```
